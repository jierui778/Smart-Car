# 硬件选型

- 主控芯片：RT1064，主流ARM架构，易用性强一些；600M速度相对于其它主频低的单片机来说有优势
- 电机驱动:DRV8701,省去BUCK升压电路,体积更小
- 电池：格式3s锂电池，放电倍率大
- 电源芯片选型:SY8303
- 摄像头选型:逐飞的总钻风
- 电机待定:380有刷电机
- 编码器:逐飞1024线正交编码器
